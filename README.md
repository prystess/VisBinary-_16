# VisBinary —— 二叉树遍历可视化系统

VisBinary 是一个用于演示与学习二叉树遍历过程的终端程序。  
本版本**移除所有 EasyX 图形库功能，仅保留逻辑实现与遍历输出**，适合数据结构课程实验使用。

项目提供多种树结构构造方式（前序带空节点、层序数组），并实现了 **递归遍历、非递归遍历、层序遍历、中序线索化遍历** 等完整功能。

---

## ✨ 功能特性

### 🌲 二叉树构造
- **前序带 # 构造**（如：`ABD##E##C#F##`）
- **层序数组构造**（如：`A B C D # E # # # F #`）

### 🔁 遍历算法（含递归与非递归）
- **前序遍历**（Preorder）
- **中序遍历**（Inorder）
- **后序遍历**（Postorder）
  - 单栈实现
  - 双栈实现
- **层序遍历**（Level Order）

### 🧵 线索二叉树
- 中序线索化
- 中序线索遍历（Threaded Inorder Traverse）

### 🧰 内存管理与结构支持
- 手动申请 / 释放节点
- 内置栈、队列结构（用于非递归与层序）
